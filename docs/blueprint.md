# **App Name**: SweetDispatch

## Core Features:

- New Request Creation: Create new customer requests with product, size, flavor, and extra selections, like a shopping cart. Collect customer name and desired delivery date/time.
- Calendar Request View: View daily request summaries (totals by product-flavor-size-extras) in a calendar-like view.  Also, see the list of requests for that day with brief details.
- Customer Request View: View pending and fulfilled requests filtered by customer.
- Request Status Management: Manage request statuses: 'requested', 'prepared', and 'fulfilled'.
- Configurable Items: Configure products, sizes, flavors, and extras with pricing, and establish valid relations (e.g., product-size, product-flavor, product-extras).
- Theme Support: Support light and dark themes.
- Localization: Multi-language support for Spanish and English.

## Style Guidelines:

- Primary color: Muted coral (#E9967A), reminiscent of baked goods, offering a warm and inviting feel.
- Background color: Very light peach (#FAF0E6), providing a soft and unobtrusive backdrop that allows content to stand out. 
- Accent color: Soft lavender (#D8BFD8), adding a touch of sophistication without overwhelming the overall warm palette.
- Body and headline font: 'PT Sans' (sans-serif) for a modern look with a touch of warmth.
- Simple, line-based icons for product types, sizes, flavors and extras.
- Clean and intuitive layout for easy navigation and order creation.
- Subtle transitions and animations on button presses to improve the user experience.
